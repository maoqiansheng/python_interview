1、获取各种变量的值

整数：{{my_num + 20}}

字符串：{{my_str + "python"}}

元组：{{my_tuple}}, 元组第0个元素：{{my_tuple[0]}}

列表：{{my_list}}, 列表第0个元素：{{my_list[0]}}

字典：{{my_dict}}, name：{{my_dict.name}}, age:{{my_dict.age}}

2、遍历元组中所有的元素

{% for item in my_tuple %}
  • {{item}}
    • {% endfor %}

    3、遍历取出列表中的偶数

    {% for item in my_list %} {% if item % 2 == 0 %} {{item}} {% endif %} {% endfor %}

    4.遍历字典内容

    {% for key in my_dict %} {# 如果直接是mydict.key ,那么这个key是一个字符串, 如果是 mydict[key], 那么key当成变量 #}
  • {{ key }} = {{my_dict[key]}}
    • {% endfor %} {# 模版自带过滤器#}

    禁用转义：safe

    {{ 'hello' | safe }}

    把变量值的首字母变大写

    {{my_str | capitalize}}

    把变量值的字母变大写

    {{my_str | upper}}

    把变量值的字母变小写

    {{'HELLO' | lower}}

    title：把值中的每个单词的首字母都转成大写

    {{'hello world python' | title}}

    reverse :字符串反转

    {{'helloworld python' | reverse}}

    format

    {{'%s is %d' | format('name', 18)}}

    striptags：渲染之前把值中所有的HTML标签都删掉

    {{ 'hello' | striptags }}

    first：取第一个元素, last:最后一个元素, length:长度, 和：sum, 排序sort

    {{my_list | first}}

    {{my_list | last}}

    {{my_list | length}}

    {{my_list | sum}}

    {{my_list | sort}}
